Rede Social - API

Documentação do serviço RESTful que gerencia usuários, posts, comentários e likes.

Usuários

Método Exemplo Descrição Resultado
GET GET /usuarios Retorna a lista com todos os usuários cadastrados. [{ "id":1, "login":"usuario", "nome":"Nome", "img":null }, ...]
POST POST /usuarios Cadastrar novo usuário. Requer login, senha e nome no corpo da requisição. O parâmetro img é opcional { "id":1, "login":"usuario", "nome":"Nome", "img":null }
POST POST /usuarios/login Efetua login. Requer login e senha no corpo da requisição. { "msg":"Login efetuado com sucesso", "usuario":{...} }

Posts

Método Exemplo Descrição Resultado
POST POST /posts Criar um post. Requer usuario, texto e/ou img no corpo da requisição. { "id":1, "usuario":1, "texto":"...", "img": "http://...." }
GET GET /posts Retorna todos os posts com seus respectivos comentários e likes [ { "id":1, "usuario":1, "texto":"...", "img": "http://....", "likes" : [...], "comentarios" : [... ] }, ... ]
GET GET /posts/10 Busca um post por ID (por exemplo, 10) { "id":10, "usuario":1, "texto":"...", "img": "http://....", "likes" : [...], "comentarios" : [... ] }
PUT PUT /posts/10 Atualiza texto e/ou img de um post por ID (por exemplo, 10). Requer usuario, texto e/ou img no corpo da requisição. Somente o usuário proprietário pode fazer a operação. { "id":10, "usuario":1, "texto":"...", "img": "http://....", "likes" : [...], "comentarios" : [... ] }
DELETE DELETE /posts/10 Deleta um post por ID (por exemplo, 10). Requer usuario no corpo da requisição. Somente o usuário proprietário pode fazer a operação. { "msg":"Post deletado com sucesso!" }

Comentários

Método Exemplo Descrição Resultado
POST POST /comentarios Adiciona um comentário. Requer usuario, post e texto no corpo da requisição. { "id":1, "usuario":1, "texto":"...", "post":5 }
PUT PUT /comentarios/33 Altera o texto de um comentário por ID (por exemplo, 33). Requer usuario e texto no corpo da requisição. Somente o usuário proprietário pode fazer a operação. { "id":33, "texto":"novo texto", ... }
DELETE DELETE /comentarios/22 Remove um comentário por ID (por exemplo, 22). Requer usuario no corpo da requisição. Somente o usuário proprietário pode fazer a operação. { "msg":"Comentário removido com sucesso" }

Likes

Método Exemplo Descrição Resultado
POST POST /likes Adiciona ou remove um like de um post específico. Requer usuario e post no corpo da requisição. { "msg":"Like adicionado com sucesso!" } ou { "msg":"Like removido com sucesso!" }

Todas as requisições aceitam dados em application/x-www-form-urlencoded
