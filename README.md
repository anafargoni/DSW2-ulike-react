# 🌐 Ulike - Rede Social

Bem-vindo ao **Ulike**, uma rede social desenvolvida como projeto fullstack.

## ⚠️ Aviso importante

Algumas ações no sistema podem **demorar alguns segundos para serem processadas**, especialmente:

- Criar posts
- Editar posts
- Excluir posts
- Carregar usuários e feed

Isso acontece por conta da comunicação com o servidor hospedado e do tempo de resposta da API.

👉 Se alguma ação não parecer acontecer imediatamente, **aguarde um pouquinho antes de tentar novamente**.

## 🚀 Link do projeto

Acesse o sistema rodando aqui:  
https://dsw2-ulike-react.vercel.app

## 💡 Observação

Este projeto utiliza:
- Frontend em React
- Backend em Node.js (Express)
- Banco de dados PostgreSQL (Neon)
- Deploy no Vercel e Render
